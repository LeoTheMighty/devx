## Goal

Every `/devx` PR carries a self-contained static review tour; human input
returns via PR comments. Per `v2/03-review-tour.md`.

## Acceptance criteria

- [ ] `devx tour build <hash>`: gather (local `git diff <base>...`, spec
      Goal/ACs, commits) → validate agent-emitted `tour.json` against the
      schema (retry protocol) → render single-file `tour.html`
      (`.devx-cache/tours/<hash>/`).
- [ ] `tour.html` template: adapted code-review-tour UI — inline JSON data
      island (`</script>` escaped), vendored diff2html JS+CSS + marked from
      node_modules, NO Mermaid (trail step tables only), system font stacks,
      zero network requests (asserted by a test scanning for `http(s)://` in
      src/href attributes).
- [ ] Tour model preserved: weight classes, dependency-ordered stops with
      must/should/skim + 4 flags, decision ledger, grep-verified trails
      (unverified edge → 🕳), blast radius incl. callers-not-updated,
      coverage rows seeded from spec ACs, `path:line` linkification →
      Total Diff deep links. No severity verdicts.
- [ ] Comment path: localStorage scratchpad + per-comment "Copy as PR
      comment" (`` `path:line` — text ``) + "Export review → clipboard".
      No server anywhere.
- [ ] `devx tour publish <hash>`: commits to orphan `devx-tours` branch
      (fetch-rebase-retry race-safe); `devx tour prune` retention.
- [ ] `pr-body`: `## 🗺 Review tour` section — htmlpreview link + raw
      fallback + orientation `<details>` markdown. Tour failures are
      fail-soft (PR opens regardless; stderr note).
- [ ] `/devx` Phase 7.5 wired (skill edit, user-foreground): build + publish
      + link before PR open.
- [ ] `/devx address <pr>`: comment→anchor mapping, in-scope fixes on the
      branch, per-comment replies, out-of-scope → debug/test specs; `devx:
      hold` + requested-changes check added to the merge tail (D-5).
- [ ] Schema/template drift-pin test (dvx107 pattern).
- [ ] Proof AC (S-2): this PR itself ships with a working tour; Leo reviews
      it via the tour.
- [ ] Full suite green.
